# Instagram Clone

This project is a work in progress. Check out the [demo](https://instagram-murtvz.vercel.app/) and [backend](https://github.com/murtvz/instagram-server) repo.

## Built with

- React
- Tailwind
- Apollo Client

## Features

- Authentication
- Following a user
- Liking a post
- Commenting on a post
- Creating new post (image url)
